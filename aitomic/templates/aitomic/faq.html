 {% blocktrans %}General info{% endblocktrans %}

{% blocktrans %}What is Aitomic?{% endblocktrans %}

{% blocktrans %}We are a marketplace where users can upload their Artificial Intelligence models, and sell them through API calls. {% endblocktrans %}

{% blocktrans %}What can I use Aitomic for?{% endblocktrans %}

{% blocktrans %}

As a provider, you can use our platform to sell your models and reach thousands of customers.


As a consumer, you can use Aitomic to improve your business, or for personal use. You can buy API calls for every model that you're interested in, and get instant access.{% endblocktrans %}

{% blocktrans %}Do I need to programming knowledge in order to use the models?{% endblocktrans %}

{% blocktrans %}You don't! We have worked hard to make Aitomic within reach of everyone, no matter if you're a programmer or not. You can use your models through our intuitive interface, where we have implemented an API system so everyone can use the AI models.{% endblocktrans %}
 {% blocktrans %}Models & interface{% endblocktrans %}

{% blocktrans %}Where can I see my models?{% endblocktrans %}

{% blocktrans %}You can find them in your profile. Go to your profile and then click on "My models". There you will find a list of the models you've uploaded.{% endblocktrans %}

{% blocktrans %}I want to upload a model.{% endblocktrans %}

{% blocktrans %}Simply go to Upload model and start filling in the form. If you have any doubts, please read our user guide.{% endblocktrans %}

{% blocktrans %}I'm trying to upload a model, but it doesn't work.{% endblocktrans %}

{% blocktrans %}Is the form showing any errors? Perhaps you didn't fill in all the fields, or uploaded the wrong file. Please read our user guide, where you will find a comprehensive tutorial of how to upload a model.{% endblocktrans %}

{% blocktrans %}Where can I edit my info?{% endblocktrans %}

{% blocktrans %}In your profile, you will find a button that says "Edit profile". There you can edit all your personal info.{% endblocktrans %}
 {% blocktrans %}Prices & revenue{% endblocktrans %}

{% blocktrans %}I want to upload a model. Can I set my own price?{% endblocktrans %}

{% blocktrans %}Yes! In Aitomic, you can set your own price for each model that you upload. Simply fill in the "Price" field in the Upload form: you can set how much you will earn for every 100 calls your model receives.{% endblocktrans %}

{% blocktrans %}How will I earn money?{% endblocktrans %}

{% blocktrans %}Every time a user makes a call to your model, you will earn the corresponding portion of the price that you set when you uploaded the model. Easy!{% endblocktrans %}

{% blocktrans %}Can I buy a model?{% endblocktrans %}

{% blocktrans %}No, sorry. What you can do is buy calls for a model. Our models are all accessed through API calls.{% endblocktrans %}

{% blocktrans %}How do I buy calls for a model?{% endblocktrans %}

{% blocktrans %}Go to the model's page, and choose "Buy". You must be registered to do this. There, you can buy however many calls you need!{% endblocktrans %}

{% blocktrans %}Do I have to pay to try a model?{% endblocktrans %}

{% blocktrans %}If the provider has selected "free trial", you don't! You can make 10 calls to the API for free, so you can test how it works before buying it.{% endblocktrans %}
 {% blocktrans %}Privacy & copyright{% endblocktrans %}

{% blocktrans %}If I upload a model, is it still mine?{% endblocktrans %}

{% blocktrans %}Always. We don't take away any right over the model. Please read our legal terms for more information regarding copyright.{% endblocktrans %}

{% blocktrans %}Can other users see my code?{% endblocktrans %}

{% blocktrans %}No. Your model and its code are and will always be yours. Users can only access your model through API calls. Please read our legal terms for more information regarding copyright.{% endblocktrans %}
