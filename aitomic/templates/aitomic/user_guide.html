{% blocktrans %}When uploading a model to Aitomic, you will need to take into consideration the following technical aspects. Please read carefully this guide before filling in the upload form.{% endblocktrans %}


{% blocktrans %}Language{% endblocktrans %}

{% blocktrans %}Our system only accepts Python as programming language. Specifically, our system supports development through version 3.6.5 of Python. We are not liable for bad functioning of the models if a different version is used.{% endblocktrans %}

{% blocktrans %}Deployment{% endblocktrans %}

{% blocktrans %}The model will be deployed in a Debian machine, with the following libraries installed by default:{% endblocktrans %}
  • Keras 2.2.4
  • Tensorflow 1.12.0
  • Pandas 0.24.2
  • Numpy 1.16.2
  • Pillow 5.4.1
  • Scipy 0.18.1
  • Scikit-learn 0.20.3

Library

{% blocktrans %}The library supported by our system for model creation will be Keras, with Tensorflow as back-end technology.{% endblocktrans %}

{% blocktrans %}Model build & upload{% endblocktrans %}

  • {% blocktrans %}Avoid using the Keras version offered by Tensorflow. Aitomic is not responsible for errors occasionated by different versions to the previously specified.{% endblocktrans %}
  • {% blocktrans %}Avoid using layers like InputLayer or Lambda.{% endblocktrans %}
  • {% blocktrans %}Avoid using Initializers.{% endblocktrans %}
  • {% blocktrans %}Avoid using custom layers in Keras.{% endblocktrans %}
  • {% blocktrans %}Export your model in json or yml format, and your weights in h5.{% endblocktrans %}
  • {% blocktrans %}Use the draft mode to test your model. If there is any issue, the output will be a message with the exception that will help you solve the problem. Don't upload your model until you're fully satisfied with its performance, since you won't be able to modify the structure or weights after it's uploaded.{% endblocktrans %}
  • {% blocktrans %}Make your models more attractive to customers with precision measures. Explain clearly the system inputs and outputs, to avoid confusion. Use the comments to resolve doubts and answer questions, and to receive feedback from users.{% endblocktrans %}
  • {% blocktrans %}In the technical description field, explain your model's context clearly, and specify the data with which your model has been trained.{% endblocktrans %}
